# Titanic Data Set

En este projecto se utiliza un conjunto de datos en donde se tienen registro de al menos el 80% de los pasajeros sin tomar en cuenta a los tripulantes. 

*Fuente*: Hind, Philip.  "Encyclopedia Titanica."  Online.  Internet.
n.p.  02 Aug 1999.

La principal pregunta que se busca resolver en este projecto es la cantidad de personas sobrevivientes respecto al total de la población y cuantas de ellas eran mujeres y cuantos hombres así como la edad en la que se encuentran la mayor cantidad de sobrevivientes y, por último, el nivel socioeconómico de los sobrevivientes utilizando la columna en donde se almacena la tarifa pagada por cada uno de los pasajeros.

# Advertising Data Set

En este conjunto de datos tenemos el gasto en publicidad hecho en distintos medios como *TV*, *Radio* y *Newspaper*, en este proyecto lo que se busca es encontrar cuales son los mejores medios para hacer publicidad para así tener mejores ventas. Para eso, nuestro conjunto de datos lo dividiremos en dos (usando la regla del 80-20), en donde para nuestro conjunto de entrenamiento usaremos el 80% de los datos y para validar nuestro modelo utilizaremos el 20% restante.

Veremos al final si solamente es conveniente mantener un solo medio de publicidad o una combinación.
